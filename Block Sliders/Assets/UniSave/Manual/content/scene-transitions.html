Loading Screen

While you’re loading a scene or save file, you can display your own static loading screen image. When you want to load a save file, you can pass in the name of your custom loading screen by calling SaveManager.Load(string saveName, string loadScreen) when loading a save file, or SaveManager.LoadLevel(string sceneName, bool isPersistent, string loadScreen) when loading a scene directly. You have to make sure that this file is in a Resources folder, or else it cannot be loaded.

 

Note that calling SaveManager.LoadLevel(string saveName, bool isPersistent, string loadScreen), with isPersistent set to false (no MultiScene Persistency) is the same as calling Application.LoadLevel(string name), but with a custom loading screen.

 

Fade Effect

Aside from a custom loading screen image, you can also choose to display UniSave’s built-in fade effect. This will fade out the screen before loading, and fade in when loading is finished. SaveManager.LoadLevel(string sceneName, bool isPersistent, float fadeInTime, float fadeOutTime) and SaveManager.Load(string saveName, float fadeInTime, float fadeOutTime). The values for fadeInTime and fadeOutTime determine (in seconds) how long it takes for the screen fade in and fade out.
